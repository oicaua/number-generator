Olá!
O projeto "Gerador de Números" é um app simples que gera números específicos que são escolhidos pelo usuário, usando os parâmetros de número mínimo, número máximo, quantidade de números e a opção de números repetidos.

Foram usados alguns imports além dos usados no HTML.
- jQuery: npm install jquery (foi usada a versão 3.6.0)

Obs: caso foi usado algum outro import, mas no foi citado aqui, será atualizado no futuro.
